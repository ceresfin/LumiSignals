## Pattern Recognition

**Quarter Level Endings:**
- .000 (Major quarters)
- .250 (Quarter points)
- .500 (Half points)
- .750 (Three-quarter points)

**Bodyguard Levels:**
For each quarter level, bodyguards are positioned 75 pips (0.0075) above and below:
- Quarter Level - 0.0075 = Lower Bodyguard
- Quarter Level + 0.0075 = Upper Bodyguard

**Examples:**
- Quarter 1.2250: Bodyguards at 1.2175 and 1.2325
- Quarter 1.2500: Bodyguards at 1.2425 and 1.2575
- Quarter 1.2750: Bodyguards at 1.2675 and 1.2825

## Usage Notes

- This list covers all major currency pairs (non-JPY)
- For JPY pairs, multiply all values by 100 (e.g., 1.2250 becomes 122.50)
- Quarter levels are static and never change
- Use this reference to avoid calculation errors
- All AI systems should reference this table rather than calculating quarter levels
