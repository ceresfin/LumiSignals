## Pattern Recognition

**JPY Quarter Level Endings:**
- .00 (Major quarters)
- 2.50 (Quarter points)
- 5.00 (Half points)
- 7.50 (Three-quarter points)

**JPY Bodyguard Levels:**
For JPY pairs, bodyguards are positioned 75 pips (0.75) above and below each quarter level:
- Quarter Level - 0.75 = Lower Bodyguard
- Quarter Level + 0.75 = Upper Bodyguard

**Examples:**
- Quarter 122.50: Bodyguards at 121.75 and 123.25
- Quarter 125.00: Bodyguards at 124.25 and 125.75
- Quarter 127.50: Bodyguards at 126.75 and 128.25

## Common JPY Pairs

## Usage Notes

- This list covers all major JPY currency pairs
- JPY pairs use 2 decimal places (vs 4 for non-JPY pairs)
- Bodyguard distance is 0.75 (75 pips) for JPY pairs
- Quarter levels are static and never change
- Use this reference to avoid calculation errors
- All AI systems should reference this table rather than calculating JPY quarter levels

## Conversion from Non-JPY Pattern

To convert non-JPY quarter levels to JPY:
- Non-JPY: 1.2250 → JPY: 122.50
- Non-JPY: 1.2500 → JPY: 125.00
- Non-JPY: 1.2750 → JPY: 127.50

The pattern is identical, just multiply by 100 and use 2 decimal places instead of 4.
